## ADDED Requirements

### Requirement: 步骤节点携带入口函数代码

AI 编排时，每个步骤节点（Layer 2）的 `code` 字段 SHALL 设置为该步骤入口函数的完整代码，而非空字符串。

#### Scenario: 步骤节点展示入口函数代码

- **WHEN** 用户点击主动线下的步骤节点
- **THEN** 右侧面板显示该步骤的描述卡片和入口函数的完整代码

### Requirement: 代码中函数名可点击跳转

HTML 渲染的代码区域中，树中已存在的函数名 SHALL 被渲染为可点击链接。

#### Scenario: 函数名显示为链接

- **WHEN** 代码中出现的函数名与树中某个节点的 label 匹配
- **THEN** 该函数名显示为带有链接样式的可点击元素

#### Scenario: 不存在于树中的函数名

- **WHEN** 代码中出现的函数名不存在于树的节点中
- **THEN** 该函数名保持普通文本样式，不可点击

### Requirement: 点击函数链接导航到目标节点

点击代码中的函数链接 SHALL 执行以下导航行为：展开目标节点的所有父节点、高亮目标节点、在右侧面板显示目标节点的代码和描述。

#### Scenario: 跳转到子动线中的函数

- **WHEN** 用户在步骤代码中点击一个函数名链接
- **THEN** 左侧树展开至目标函数节点并高亮，右侧面板更新为目标函数的代码和描述

#### Scenario: 跳转后树面板滚动

- **WHEN** 目标节点不在左侧树面板可视区域内
- **THEN** 树面板自动滚动使目标节点可见
